昨天 Anthropic 发了 Claude 4，包括 Sonnet 和 Opus 两个版本，Sonnet 直接开放使用，Opus 后续放出。

## 评测数字

从 Anthropic 放出的数据来看，Claude 4 Sonnet 在代码和推理类任务上的跑分超过了 GPT-4o 和 Gemini 2.5 Pro，和 o3 在某些任务上有的来有的输。

当然跑分和实际使用是两回事，先记着，等用了再说。

## 代码方向的重点押注

Anthropic 这次明显在强调 Claude 4 的代码能力，专门列了 SWE-bench 的成绩（这是测模型能不能解决真实 GitHub issue 的评测），说是大幅提升。

结合之前 Claude 在 Cursor、Continue 这类编程工具里的整合越来越多，感觉 Anthropic 的策略是：写代码是他们的核心战场。

## 价格

API 价格和 Claude 3.7 相比略有调整，Sonnet 定价在市场上算中等，没有特别贵。

## 第一印象

今天试了一下，问了几个技术问题，感觉回答的结构确实比 3.7 更好，不太绕，有点"直接说重点"的感觉。

代码方面需要更认真测一测，之后写专门的对比帖。
