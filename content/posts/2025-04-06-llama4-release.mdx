昨天 Meta 发了 Llama 4，两个版本：Scout 和 Maverick。又是开源大模型的一次重要发布。

## 这次发了什么

**Scout**：轻量版，适合在边缘设备或者资源有限的环境运行，多模态支持（能处理图片）。

**Maverick**：更大的版本，性能更强，在部分评测上达到了 GPT-4o 的水平，同样支持图文输入。

都是开源的，可以本地部署。

## 多模态这次是重点

之前 Llama 3.x 系列主要是文字，这次 Llama 4 把图像理解放进来了，是一次不小的跨越。

对于需要处理图文内容但又不想用闭源 API 的场景，Llama 4 Maverick 会是一个很有吸引力的选项。

## 开源生态在加速

去年用 Llama 3.1 的时候，开源模型和闭源模型的差距还比较明显。现在 Llama 4 Maverick 据说跑分已经接近 GPT-4o 了。

差距在缩小，而且 Meta 把开源当成战略一直在投入。对整个生态来说是好事，各家 API 的价格竞争压力也会更大。

有兴趣的话 Maverick 可以通过几个平台 API 调用，之后找时间测测多模态的效果。
