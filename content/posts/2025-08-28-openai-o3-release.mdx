OpenAI 的 o3 昨天正式开放了，之前只有申请才能用，现在 ChatGPT Plus 用户可以直接选。

## o3 是什么

o3 是 OpenAI 推理模型系列（o1 → o1 pro → o3）里目前最强的，主打需要大量推理步骤的任务：数学竞赛题、高难度代码、科学问题等等。

它的特点是：给它更多"计算时间"（token budget），它能给出更准确的答案。你可以设置它"想多久"，想得越久结果越好，但也越贵越慢。

## 用来干什么最合适

o3 不是用来替代日常对话的，用来问"帮我写封邮件"有点浪费。

更适合的场景是那种你自己解不了或者用普通模型得到了错误答案的硬题——复杂算法题、数学证明、需要多步骤推理的分析。

## 价格

API 按 token 计费，比 o1 贵，比 GPT-4o 贵很多。所以不是日常用的模型，是"需要时才请出来"的那种。

## 感受

今天用 o3 解了一道之前用 Claude 4 没解对的题，o3 的推理过程写得很清楚，最终答案也是对的。

但等的时间确实长，感觉像是找了一个很认真但有点慢的专家。某些情况下值这个等待，某些情况下不值。
