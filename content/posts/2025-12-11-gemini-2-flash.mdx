昨天 Google 发了 Gemini 2.0 Flash，这是 Gemini 2.0 系列里的轻量快速版本。

## 有意思的点：API 免费

这次 Gemini 2.0 Flash 的 API 有免费额度，而且限制不算低。对于做小项目或者个人开发者来说，用这个 API 成本几乎为零。

和 OpenAI、Anthropic 的 API 定价比，这是一个很明显的差异化策略。

## 速度

"Flash"这个名字不是随便起的，响应速度确实快，比同级别的模型明显更快。

用来做需要实时反馈的应用（对话流式输出、实时语音等），速度优势很明显。

## 能力够不够

Flash 版本不是最强的模型，在难度高的推理任务上会比 Pro 版差。

但对于大多数实用场景——总结文章、回答问题、辅助写作、处理结构化数据——Flash 的能力完全够用，用不着掏大模型。

## 今年年底的发布节奏

最近各家年底都在密集发布，年底好像是 AI 圈的"冲刺季"。

Gemini 2.0 Flash 的免费 API 对个人项目来说是个好消息，打算拿来做一个小工具试试。
