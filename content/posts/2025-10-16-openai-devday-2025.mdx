OpenAI DevDay 昨天在旧金山举行，这个活动主要面向开发者，发布的东西偏技术层面，没有太多消费者端的大新闻。

## 最大的发布：Agents SDK

OpenAI 推出了一套 Agents SDK，让开发者更方便地构建 AI 智能体（Agent）——能自主完成多步骤任务的程序，比如自动浏览网页、调用工具、执行代码、写文件。

这个 SDK 封装了很多之前要自己处理的细节：如何让模型调用工具、如何处理多步骤的对话状态、如何处理错误和重试。

对于想构建 Agent 类应用的开发者，这个工具库会省很多工夫。

## 其他发布

**Realtime API 升级**：语音交互的延迟更低，支持更多语言，情绪感知更好。

**Fine-tuning 改进**：小样本微调的效果提升，让更多公司能用自己的数据调整模型。

**价格调整**：部分 API 的价格降了，GPT-4o 的调用成本比年初便宜了一大截。

## 感受

这次 DevDay 的主题词是"Agents"，OpenAI 显然在押注 Agent 是下一波爆发点。

不过我自己现在还没有深度开发 Agent 应用的需求，这些工具先了解着，等有具体项目需要再细看。

价格降这件事是真实惠，之前 API 调用成本是很多人的障碍。
